Generate an SSH key.

1)Add it to your GitHub account.

Step 1: Generate an SSH key

Linux & macOS (Terminal):

ssh-keygen -t ed25519 -C "your_email@example.com"

Windows (PowerShell or Git Bash):

ssh-keygen -t ed25519 -C "your_email@example.com"

Press Enter to accept the default location (~/.ssh/id_ed25519).

Optionally add a passphrase, or press Enter for none.

Step 2: Start SSH agent & add the key

Linux/macOS:

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

Windows (PowerShell with Git installed):

eval $(ssh-agent -s)
ssh-add ~/.ssh/id_ed25519

Step 3: Copy the public key

Linux/macOS:

cat ~/.ssh/id_ed25519.pub

Windows (PowerShell):

type ~\.ssh\id_ed25519.pub

Copy the whole line starting with ssh-ed25519.

Step 6: Use SSH instead of HTTPS

If you already cloned with HTTPS, switch to SSH:

git remote set-url origin git@github.com:USERNAME/REPO.git

If you want to freshly clone with SSH:

git clone git@github.com:USERNAME/REPO.git

Step 7: Push and Pull with SSH

Once your repo is set to SSH:

git pull origin main
git push origin main
